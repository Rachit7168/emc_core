# EMC Cafe Website – UI Redesign Concept

This repository contains a **modern redesign concept** for the EMC Cafe website.

Original Website:
https://emcsurat.com

The goal of this redesign is to improve:

* Visual design
* User experience
* Mobile responsiveness
* Modern UI components
* Overall brand presentation

## Improvements Made

* Cleaner and modern layout
* Improved typography
* Better color balance
* Organized sections
* More modern landing page structure
* Improved visual hierarchy

## Purpose of this Project

This redesign was created as a **UI/UX improvement concept** for the EMC Cafe website.

If the EMC Cafe team finds the redesign useful, I would be happy to collaborate to help improve the live website.
